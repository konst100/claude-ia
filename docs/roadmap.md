# Роадмап: От новичка до AI-разработчика для бизнеса

**Цель:** За 3-4 месяца (10-20 ч/неделю) построить один серьёзный учебный проект, который:
- Научит работать со всеми ключевыми ИИ-инструментами 2026 года
- Покажет современную архитектуру и процесс разработки
- Станет твоим главным портфолио для собеседований
- Будет реальным продуктом, который могут использовать бизнесы

---

## Главный проект: "AI Ops Hub"

Веб-платформа для малого/среднего бизнеса, которая объединяет несколько AI-функций в одном месте:

1. **AI-ассистент для клиентов** (чат-бот на сайт)
2. **Документ-Q&A** — загрузил договоры/инструкции → задаёшь вопросы (RAG)
3. **AI-агенты** для рутины — драфт писем, резюме встреч, квалификация лидов
4. **Workflow-автоматизация** — связь с Gmail, Calendar, Slack, Notion
5. **Дашборд** — аналитика использования ИИ, экономия времени

Почему именно такой проект: на рынке 2026 года ровно это и продают AI-консультанты бизнесу. Если ты сделаешь это сам — у тебя на руках готовый кейс для любой вакансии "AI Engineer / AI Developer / Automation Specialist".

       ┌──────────────┼──────────────┬──────────────┐
       ▼              ▼              ▼              ▼
┌──────────┐  ┌─────────────┐  ┌──────────┐  ┌──────────┐
│ AI ROUTER│  │ VECTOR DB   │  │ DATABASE │  │ WORKFLOWS│

Это **современная стандартная архитектура AI-приложения**. Её используют OpenAI, Anthropic, Perplexity, Notion AI и сотни стартапов.

## Какой ИИ выбрать?

Короткий ответ: **не один, а все сразу через "AI-роутер"**. Это ключевой профессиональный навык 2026 года.

| ИИ | Где используется в проекте | Сильные стороны |
|---|---|---|
| **Claude (Anthropic)** | Длинные документы, рассуждения, написание кода | Лучший по качеству, длинный контекст, безопасность |
| **GPT-4o (OpenAI)** | Универсальные задачи, tool calling, мультимодальность | Быстро, много готовых библиотек, function calling |
| **Gemini (Google)** | Дёшево, очень быстро, картинки/видео | Цена/скорость, нативные мультимедиа |
| **Whisper / ElevenLabs** | Голос (распознавание/синтез) | Стандарт де-факто |
| **Локальные модели (Ollama)** | Приватные данные клиента | Не уходит в облако |

В коде ты создашь "router" — функцию, которая выбирает модель под задачу. Это даст резюме-строку: *"Multi-provider AI architecture with cost optimization"*.

## Современный стек (всё, что ты выучишь)

**Языки и фреймворки:**
- TypeScript + Next.js 15 (App Router) — стандарт фронтенда
- Python — для AI-скриптов и data processing
- Tailwind CSS + shadcn/ui — UI

**AI-инструменты:**
- Anthropic SDK, OpenAI SDK
- LangChain / LlamaIndex (агенты, RAG)
- Vercel AI SDK (стриминг)
- MCP (Model Context Protocol) — то, чем мы сейчас пользуемся в Cowork

**Инфраструктура:**
- Vercel (хостинг)
- Supabase (БД + auth + storage)
- Pinecone или pgvector (векторная БД)
- Stripe (платежи — для готового SaaS)

**Dev-tools:**
- Claude Code / Cursor (AI-программирование)
- Git + GitHub (обязательно)
- n8n (low-code автоматизации)
- Vercel Analytics + Sentry

## Принципы работы (как не сгореть)

1. **Vibe-coding — ОК, но понимай что происходит.** Claude Code напишет тебе код, но КАЖДЫЙ раз спроси: "объясни что ты сделал и почему". Через месяц будешь читать код сам.

2. **Деплой каждую неделю.** Даже если фича маленькая — выкатывай. Это формирует профессиональную привычку.

3. **Git коммиты часто.** Любая логически завершённая правка → коммит. Это будет твоя история работы.

4. **Документируй для себя.** В каждой папке — README с тем, что и зачем. Через 2 месяца будешь благодарен.

5. **Не залипай в выбор инструмента.** Если сомневаешься между двумя — бери первый. Время важнее идеального выбора.
